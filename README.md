_MacOS clipboard app for developers_

- Snippet language detection
- Snippet syntax highlighting
- Supports all [major languages](#supported-languages)
- Customizable / importable [themes](#theming)
- Blazingly fast search
- Secure. 100% local storage + ML models
- Keyboard centric actions

### Theming

Several built-in themes are provided:
- One Dark
- Dracula
- Github Dark
- Gruvbox
- Monokai
- Solarized Dark
- Solarized Light

Additionally, Zettl supports importing any valid .tmTheme file
